# Facial Reaction Recognition System

Designed a Facial Recogntion System that used and test multiple statistical approaches such as SVM (Support Vector Machines), PCA (Principal Component Analysis), and LDA (Linear Discriminant Analysis). Then compared various result to se which classifier performed the best. 

The dataset consited of 7 different facial Reactions
	1. Angry
	2. Disgust
	3. Fear
	4. Happy
	5. Sad
	6. Surprise
	7. Neural
